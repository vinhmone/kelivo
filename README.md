## ✨ Tính năng

- 🎨 **Thiết kế hiện đại** - Ngôn ngữ thiết kế Material You với hỗ trợ chủ đề màu động (Android 12+).
- 🌙 **Chế độ tối** - Chủ đề tối được tối ưu để bảo vệ mắt của bạn.
- 🌍 **Hỗ trợ đa ngôn ngữ** - Hỗ trợ giao diện Tiếng Việt, Tiếng Anh và Tiếng Trung.
- 🖥️ **Hỗ trợ đa nền tảng** - Di động (Android/iOS/Harmony) và máy tính để bàn (Windows/macOS/Linux).
- 🔄 **Hỗ trợ nhiều nhà cung cấp** - Hỗ trợ các nhà cung cấp AI phổ biến như OpenAI, Google Gemini, Anthropic, v.v.
- 🤖 **Trợ lý tùy chỉnh** - Tạo và quản lý các trợ lý AI cá nhân hóa.
- 🖼️ **Đầu vào đa phương thức** - Hỗ trợ nhiều định dạng như hình ảnh, tài liệu văn bản, PDF, tài liệu Word, v.v.
- 📝 **Kết xuất Markdown** - Hỗ trợ đầy đủ tô sáng mã, công thức LaTeX, bảng và nhiều nội dung khác.
- 🎙️ **Nhà cung cấp giọng nói/TTS** - Tích hợp TTS hệ thống, đồng thời hỗ trợ máy chủ giọng nói OpenAI / Google Gemini / ElevenLabs.
- 🛠️ **Hỗ trợ MCP** - Tích hợp công cụ Model Context Protocol.
- 🧰 **Công cụ MCP tích hợp** - Bao gồm công cụ MCP Fetch tích hợp sẵn.
- 🔍 **Tìm kiếm web** - Tích hợp nhiều công cụ tìm kiếm (Bing, DuckDuckGo, Exa, Tavily, Zhipu, LinkUp, Brave, Metaso, SearXNG, Ollama, Jina, Perplexity, Bocha, Serper, Grok).
- 🧩 **Biến prompt** - Hỗ trợ các biến động như tên mô hình, thời gian, v.v.
- 📤 **Chia sẻ mã QR** - Xuất và nhập cấu hình nhà cung cấp bằng mã QR.
- 💾 **Sao lưu dữ liệu** - Hỗ trợ sao lưu và khôi phục lịch sử trò chuyện.
- 🌐 **Yêu cầu tùy chỉnh** - Hỗ trợ tiêu đề và nội dung yêu cầu HTTP tùy chỉnh.
- 🔡 **Phông chữ tùy chỉnh** - Sử dụng phông chữ riêng của bạn (phông chữ hệ thống / Google Fonts).
- ⚙️ **Tạo nội dung nền trên Android** - Tiếp tục tạo nội dung trò chuyện ở chế độ nền (thiết lập tùy chọn).
